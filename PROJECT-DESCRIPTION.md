Fourth Firecrawl is a Claude plugin that turns the Firecrawl MCP into a competitive intelligence and market-research engine tuned for the hospitality technology space.
It ships task-oriented skills for competitor deep-dives, trade-press market scans, content gap analysis, and staged KB ingestion into the Fourth Marketing Brain — each backed by curated source registries and extraction schemas.
Built for Fourth Marketing team members working inside Claude Cowork or Claude Code; each user runs against their own Firecrawl API key so credit usage stays individually accountable.
